
Subir a un repo ya existente

git init
git add -A
git commit -m "Comentario"
git branch -M main (escoger branch)
git remote add origin (link)
git push -u origin main (escoger branch)

Actualizar archivos de un repo ya existe
git add -A
git commit -m "Actualizacion"
git branch -M main (escoger branch)
git remote add origin (link)
git push -u origin main (escoger branch)
